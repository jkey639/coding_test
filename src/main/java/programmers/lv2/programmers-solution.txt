MakeTwoQueueSumEqual
1.
두 큐 요소 모든 수의 합 / 2 = 각각의 큐 요소의 합
묙표 값인 각각의 큐 요소의 합과 현재 큐 요소의 합의 값을 비교하여 모자라거나 넘치는 차이를 확인
차이가 있을 경우 하나의 큐를 선택해 요소를 넘겨줌 : 값이 모자란 큐를 선택한 경우 값이 모자라지 않을 때까지 넘겨주고, 값이 넘치는 큐를 선택한 경우 값이 넘치지 않을 때까지 넘겨줌
넘겨주는 액션 후 목표 값이 도달하지 않고 반대로 모자라거나 넘치는 경우 반대의 큐를 선택해 다시 요소를 넘겨줌
다음 결과가 나올 때까지 반복 : 목표 값을 맞췄거나 원래 큐 요소 순서로 되돌아왔을 경우

* 원래 큐 요소 순서로 되돌아오는 경우?
두 큐 요소 모든 수의 합이 짝수이면 각각의 큐 요소의 합이 같을 수는 있다.
각각의 큐 요소의 합이 같지만 요소를 조합하여 그 합을 만들 수 없는 경우?? : 요소 중에 요소의 합보다 큰 수가 있을 경우
   요소 중에 요소의 합보다 모든 수가 작아도 만들 수 없는 경우가 있다 : 모든 요소 합 30, 각각 큐 합 15, 요소 14, 14, 2 >> 만들 수 없다
